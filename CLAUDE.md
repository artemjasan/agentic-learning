# Study Lab

Hands-on systems engineering lab. Deep dives into databases, distributed systems, infrastructure, security, messaging, and more — primarily in Go.

## Agent Quick Start

1. Read `AGENTS.md` — project vision, conventions, workflow
2. Read `PROGRESS.md` — what's done, what's current, what's next
3. Read `TOPICS.md` — full topic list with status and hands-on objectives

## Hard Rules

- **Go is the default language.** Exceptions: Rust for kernel work, Python for descriptors and visualization tooling.
- **Hands-on first.** Build it, run it, break it, benchmark it. Theory serves the experiment, not the other way around.
- **Update PROGRESS.md** after every completed session — what was done, what was learned, what's next.
- Each topic lives in `modules/<NN-slug>/` as a self-contained deep dive.
